# AuroraOS Keyboard Fix Notes

Version: AuroraOS Stable Build 1.5.1

Changes:
- Added basic Shift support for DE and US keyboard layouts.
- `:` now works in the shell/editor.
- Lumen editor also accepts `.help`, `.show`, `.save`, `.clear`, `.quit` as fallback commands.

Usage:
- German layout: `:` with `Shift` + `.`
- US layout: `:` with `Shift` + `;`
