# AuroraOS

AuroraOS is a completely new operating system that is not based on Linux, Windows, or any other existing operating system.

Its goal is to combine the simplicity of Windows with the modularity of Linux, while also being one of the first operating systems to integrate AI directly into its shell.

AuroraOS is not based on DOS or UNIX. Instead, it uses **AuroraFS**, a custom filesystem with UNIX-like structures.

At the moment, AuroraOS supports **German** and **English** only.

AuroraOS currently has **no graphical user interface** and is based entirely on a **CLI**.

## Project status

AuroraOS is still in a very early stage of development.  
Full functionality, stability, and compatibility are **not guaranteed**.

Bug reports, suggestions, and feedback are very welcome.
